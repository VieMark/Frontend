Инструкция:
Создайте HTML-документ и добавьте несколько блоков на страницу.
Примените к блокам стили для настройки их размеров и отступов.
Исследуйте, как свойства блочной модели влияют на отображение элементов.

Задание:
Создайте три блока на странице:
Первый блок будет с фиксированной шириной и высотой.

Второй блок с максимальной шириной (max-width) и высотой (max-height).

Третий блок с динамическими размерами (без фиксированной ширины), но с отступами (padding), границами (border) и внешними отступами (margin).


Настройте для каждого блока следующие свойства:

Width и height для фиксированных размеров.

Padding для внутренних отступов.

Border для добавления границы.

Margin для внешних отступов.

Используйте свойство box-sizing, чтобы понять разницу между значениями content-box и border-box.
